Sistema de Agendamento para Barbearia

Sistema desenvolvido em Java com o objetivo de simular um sistema de agendamento para uma barbearia, permitindo cadastrar horários, selecionar tipos de cortes e armazenar dados em arquivo CSV.

Descrição

Este projeto foi desenvolvido como prática de Programação Orientada a Objetos (POO), manipulação de arquivos e tratamento de exceções em Java.
O sistema permite realizar agendamentos de clientes, validando datas e horários disponíveis e armazenando as informações em um arquivo CSV.

Tecnologias Utilizadas

Java

Programação Orientada a Objetos (POO)

Tratamento de Exceções

Enum

Manipulação de Arquivos (CSV)

Git e GitHub

Funcionalidades

Cadastro de agendamentos

Validação de data e horário

Controle de horários disponíveis

Seleção de tipo de corte

Armazenamento dos dados em arquivo CSV

Tratamento de exceções personalizadas
